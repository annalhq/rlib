# Digital Electronics

## CAT1001

### Introduction

*Course code: CAT1001*

*Syllabus* : 

Unit-1 : Basics of Digital Electronics
Motivation for digital systems: Logic and Boolean algebra, Number Systems. Logic Gates & Truth
Tables, Demorgan's law, Minimization of combinational circuits using Karnaugh maps up to five
variables. Map manipulation-essential prime implicants, non-essential prime implicants.

Unit-2 : Combinational Circuit Design
Design procedure: Multiplexers, Demultiplexer, Encoders, Decoders, Code Converters, Adders,
Subtractor (Half, Full), BCD Adder/ Subtractor, ripple and carry look-ahead addition booth's
Algorithm, bit-pair recoding, Integer Division- restoring and non-restoring division.

Unit-3 : Sequential circuit Design-I
Storage elements, Flip-flops and latches: D, T, J/K, S/R flip-flops. Master Slave Conversion of one of
type of F/F to another Sequential circuit. Analysis-Input equations, state table, and analysis with J-K
Flip flops. Sequential circuit Design, Design procedure, designing with D & J-K Flip flop.

Unit-4 : Sequential circuit Design-II
Counters, asynchronous and synchronous design using state and excitation tables. Registers & Shift
registers., Mealey & Moore Machines.

Unit-5 : Memory & Programmable logic Devices
Semiconductor RAM memories, Static and Dynamic Memories, ROM, higher order memory design,
multi-module memories, Memory interleaving, , Secondary storage – Magnetic disk, Optical disk,
PLA, PAL.

Unit-6 : Fundamental of Microprocessor
Introduction to µp 8085, Addressing modes, Instruction set, Programming of µp 8085.
